# Triage Status

The skills speak in terms of five canonical triage roles. This file maps those
roles to the status strings used in local markdown issue files.

| Canonical role | Status in local issue | Meaning                                   |
| -------------- | --------------------- | ----------------------------------------- |
| `needs-triage` | `needs-triage`        | Maintainer needs to evaluate this issue   |
| `needs-info`   | `needs-info`          | Waiting on reporter for more information  |
| `ready`        | `ready`               | Fully specified, ready for implementation |
| `wontfix`      | `wontfix`             | Will not be actioned                      |

When a skill mentions a role, use the corresponding status string from this
table in the issue's `Status:` line.

Edit the right-hand column to match whatever vocabulary you actually use.
